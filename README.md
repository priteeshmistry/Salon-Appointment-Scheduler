Project Overview

This is a certification project for freeCodeCamp's Relational Database course. It uses a Bash script (salon.sh) to interact with a PostgreSQL database (salon) via the command line.

Technical Implementation

Database Schema: Created three tables (services, customers, appointments) linked by primary and foreign keys.

Bash Scripting: Uses case and read commands to display a menu and accept user input from the terminal.

SQL Queries: Executes psql commands directly inside the shell script to query existing customers by phone number and INSERT new appointments or users into the database.
